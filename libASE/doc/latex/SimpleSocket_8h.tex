\subsection{Detailed Description}
\hyperlink{SimpleSocket_8h}{SimpleSocket.h} provides simple socket interface as needed by \hyperlink{CommandSender_8h}{CommandSender.h} and \hyperlink{EventListner_8h}{EventListner.h} to interface with the Unified Simulator for Self-Reconfigurable Robots (USSR).
