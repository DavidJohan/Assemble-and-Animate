\subsection{Detailed Description}
\hyperlink{AbstractModuleApi_8h}{AbstractModuleApi.h} provides a generic interface or hardware abstraction layer to control many aspects of most modules.The goal is provide a method to write generic control code which can be run on different platforms. All control strategies included in the ASE library uses only this interface to access the modules.
