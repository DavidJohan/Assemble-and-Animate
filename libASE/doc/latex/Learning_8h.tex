\subsection{Detailed Description}
\hyperlink{Learning_8h}{Learning.h} a simple reinforcement learning algoritm for modular robots. Following the algorithm described in: D.J.Christensen et al. \char`\"{}Morphology Independent Learning for Modular robots\char`\"{} DARS2008 Basically each module is independently optimizing it own reward using an exponential moving average.
