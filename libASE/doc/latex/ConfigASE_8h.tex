\subsection{Detailed Description}
\hyperlink{ConfigASE_8h}{ConfigASE.h} contains defines that configures how ASE is compiled. All memory allocation is at compile time, therefore you must here specify how many instances of different classes must be allocated. E.g. how many Gradients are will the application use?
