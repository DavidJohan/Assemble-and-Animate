\subsection{Detailed Description}
\hyperlink{AtronSocketApi_8h}{AtronSocketApi.h} provides an interface for an ATRON module simulated in the Unified Simulator for Self-Reconfigurable Robots (USSR). API calls to the modules are communicated to USSR though a socket. Similar messages (asynchronous events) to the moduels are send from USSR to the module controller through a seperate socket. In total two sockets are used per module. Initially a call to atronApi\_\-setup(...) must be made. This API builds on top of \hyperlink{CommandSender_8h}{CommandSender.h} and \hyperlink{EventListner_8h}{EventListner.h}.
