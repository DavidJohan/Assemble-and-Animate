\subsection{Detailed Description}
\hyperlink{IDManager__MC_8h}{IDManager\_\-MC.h} enables a remote PC running ModularCommander to set a sofware ID on modules, by making the mapping hardwareID-\/$>$softwareID. This is done in a transparent way from the application.
