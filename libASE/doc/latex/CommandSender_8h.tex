\subsection{Detailed Description}
\hyperlink{CommandSender_8h}{CommandSender.h} provides together with \hyperlink{EventListner_8h}{EventListner.h} an generic socket based interface to the Unified Simulator for Self-Reconfigurable Robots (USSR). String formatted as function calls are send to the simulator for execution (through reflection). For example \char`\"{}SendMessage 123,32,10, 3 5\char`\"{} will trigger the SendMessage method in USSR with the appropiate parameters: the message (\{123,32,10\}), the message length (3) and the channel(5).
