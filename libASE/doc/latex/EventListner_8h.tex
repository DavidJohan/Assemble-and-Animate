\subsection{Detailed Description}
\hyperlink{EventListner_8h}{EventListner.h} provides together with \hyperlink{CommandSender_8h}{CommandSender.h} an generic socket based interface to the Unified Simulator for Self-Reconfigurable Robots (USSR). Events triggered in USSR are send to this EventListner through a socket. For example an event could be \char`\"{}HandleMessage 123,32,10, 3 1\char`\"{} which means the the module in USSR has received a message (\{123,32,10\}) of length 3 on channel 1. \hyperlink{EventListner_8h}{EventListner.h} provides an observer based interface for handlers to be notified of such events.
