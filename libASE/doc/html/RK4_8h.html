Computes an estimation of y(t+h) given y(t) by using the Runge-Kutta (RTK4) method. Needed parameters are: time t, estimation of y(t), a step size h, a function pointer to compute y'(t, y, *params). Use a void pointer to pass other parameters to call-back function.
