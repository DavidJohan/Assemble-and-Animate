Detailed Description

Generic, super class for CPG implementation. Not to be instantiated directly, but is used by sub-classes. Known sub-classes: CnoCpg.h and HopfCartesianCpg.h

Author:
David Johan Christensen, Modular Robotics Lab, AdapTronics Group, Maresk Institute, University of Southern Denmark
