)

Setup the API for USSR, two free socket ports portRC and portEvent must be provided for commands and events respectively. Also a host (usually LOCALHOST) and a pointer to a function (e.g. handleMessage(char* message, char messageSize, char channel)) that will handle messages received from other modules.
